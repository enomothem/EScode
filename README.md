## ES_T0003 介绍
本工具采用了平铺的方式，这样能把数据任意的转换，并添加了置换按钮，让解码的内容进行再一次的加密，由于零宽不好复制，添加复制按钮，点击置换按钮旁边的小按钮复制上下文本框的内容。
